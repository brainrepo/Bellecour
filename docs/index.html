Features

Everything you need,
nothing you don't

A native macOS menu bar app that sits quietly until you need it. No windows to manage, no apps to switch to.

Hold-to-Record

Press and hold the global shortcut anywhere on your Mac to start recording. Release to instantly transcribe. No clicks, no windows, works system-wide.

Instant Paste

Transcribed text is pasted directly into your active app with a single keystroke. No copy-paste dance.

AI Transformations

5 built-in transforms (grammar, formal, fun, concise, actions) plus 3 translations. Apply any with a single keypress before pasting.

Multi-Language

Transcribe in 8 languages and translate between them. Click the pill or press L to switch languages instantly.

Secure by Design

Your API key lives in the macOS Keychain, not in config files. No data stored on external servers.

Create Your Own Transforms

Write your own AI transformations with custom prompts — summarize for Slack, format as email, extract keywords, anything you need. Add, reorder, edit, or remove from Settings.

Interface

Three states,
zero friction

A tiny floating pill that expands only when you need it, then gets out of the way.

1

Idle Pill

A minimal 64×34px pill floats on your screen showing a microphone icon and your current language. Click to cycle languages, right-click for the full picker. It's always there, never in the way.

2

Recording

Hold the shortcut and the pill expands into a recording bar with a pulsing red dot, a live audio waveform, and an elapsed time counter. Release to stop and transcribe instantly.

3

Result Card

Your transcription appears in an editable card with an audio player for playback, a toolbar of AI transformations, and action buttons. Press Enter to paste, Escape to discard, or 1-9 to transform.

Get Started

Up and running
in minutes

1

Clone & Install

Clone the repo and install dependencies with pnpm.

pnpm install
2

Add API Key

Open Settings from the tray menu and enter your OpenAI API key. Stored in macOS Keychain.

3

Start Talking

Run the app and hold ctrlcmdB to record. Release to transcribe.

pnpm tauri dev

macOS Permissions

Bellecour needs Microphone access (prompted automatically) and Accessibility permission for auto-paste. Grant it in System Settings → Privacy & Security → Accessibility. If the permission seems stuck, remove the app from the list and re-add it.
